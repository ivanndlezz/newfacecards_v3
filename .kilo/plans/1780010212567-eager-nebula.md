# Plan: Implement Bottom Sheet with Example Hydration in add-button-link-experience

## Goal
Replicate the bottom sheet system from `admin/user/ivangonzalez/index.html` in `templates/buttons/add-button-link-experience/index.html` to open a bottom sheet and hydrate two different example contents.

## Steps

### 1. Analyze the Bottom Sheet System
- The bottom sheet is implemented via `/shared/components/bottom-sheet/bottom-sheet.js` which provides `constructSheet` and `closeSheet` functions.
- The CSS is auto-injected by the JavaScript module.
- Content is provided via a configuration object passed to `constructSheet`.
- Example from admin file: uses a template (`#tmpl-profile-editor`) for the content.

### 2. Determine Hydration Target
- Add a container in the body where example contents will be hydrated (e.g., `<div id="hydration-target"></div>`).
- Define two example data sets (e.g., exampleButton1 and exampleButton2) representing different button configurations (text, URL, experience description).

### 3. Create Bottom Sheet Content Template
- Add a hidden template element in the body (or head) that contains the HTML for the bottom sheet content.
- The content should include controls to select between the two examples (e.g., two buttons or a radio group).
- When an example is selected, it will trigger hydration of the target container with the corresponding data.

### 4. Add JavaScript Logic
- Add a module script to:
  - Import `constructSheet` and `closeSheet` from `/shared/components/bottom-sheet/bottom-sheet.js`.
  - Define the two example data sets.
  - Define a function to hydrate the target container with given data (e.g., set innerHTML or update specific elements).
  - Define a function to open the bottom sheet with the content template.
  - Bind the open action to a button (e.g., a floating action button or a button in the header).
  - In the bottom sheet content, bind events to the example selection controls to call the hydration function and close the sheet.

### 5. Styling Considerations
- The bottom sheet CSS is handled by the component.
- Ensure the hydration target has appropriate styling to display the example contents clearly.
- Consider adding a button to open the bottom sheet (e.g., a fixed button or in the header).

### 6. Implementation Details

#### HTML Changes
- In `<body>`:
  - Add a button to open the bottom sheet: `<button id="openBottomSheet">Open Examples</button>`
  - Add the hydration target: `<div id="hydration-target" class="hydration-target"></div>`
  - Add a template for the bottom sheet content:
    ```html
    <template id="example-bottom-sheet-template">
      <div class="bottom-sheet-content">
        <h3>Choose Example</h3>
        <div class="example-buttons">
          <button data-example="1">Example 1</button>
          <button data-example="2">Example 2</button>
        </div>
      </div>
    </template>
    ```

#### JavaScript Changes
- Add a module script (type="module") that:
  - Imports `constructSheet`, `closeSheet` from the shared path.
  - Defines example data:
    ```javascript
    const exampleData = {
      1: { text: 'Get Started', url: '#', experience: 'Beginner friendly' },
      2: { text: 'Learn More', url: 'https://example.com', experience: 'Advanced details' }
    };
    ```
  - Defines hydration function:
    ```javascript
    function hydrateTarget(data) {
      const target = document.getElementById('hydration-target');
      if (!target) return;
      target.innerHTML = `
        <div class="example-card">
          <h2>${data.text}</h2>
          <a href="${data.url}" class="example-link">Visit Link</a>
          <p class="example-experience">${data.experience}</p>
        </div>
      `;
    }
    ```
  - Defines openBottomSheet function:
    ```javascript
    function openExampleBottomSheet() {
      const template = document.getElementById('example-bottom-sheet-template');
      constructSheet({
        title: 'Example Selector',
        content: template,
        bottom_actions: {
          layout: 'single',
          primary: {
            label: 'Close',
            callback: () => closeSheet() // or use "close" string
          }
        }
      });
      // After sheet is open, bind events to the example buttons
      // Note: The bottom sheet content is now in the DOM, we can query it
      // We'll use event delegation or wait for the sheet to open.
    }
    ```
  - Add event listener to the open button:
    ```javascript
    document.getElementById('openBottomSheet').addEventListener('click', openExampleBottomSheet);
    ```
  - Handle example selection (using event delegation on the document or within the bottom sheet container):
    ```javascript
    document.addEventListener('click', (e) => {
      if (e.target.matches('[data-example]')) {
        const exampleNum = e.target.dataset.example;
        const data = exampleData[exampleNum];
        hydrateTarget(data);
        closeSheet(); // close after selection
      }
    });
    ```
